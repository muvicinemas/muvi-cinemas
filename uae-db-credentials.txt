======================================

1. Open DBeaver → Database → New Database Connection
2. Select "PostgreSQL" → Next
3. Fill in:
     Host:     uatclusterdb.cluster-cwyxrbeukelc.me-central-1.rds.amazonaws.com
     Port:     5432
     Database: muvi_main_service        ← pick the one you need
     Username: postgres
     Password: HR)q0Cpn?D$OyqREGo0-BlupVueo

4. IMPORTANT — to hide other databases:
     Click the "PostgreSQL" tab at the top
     UNCHECK "Show all databases"

5. Click "Test Connection" → should say "Connected"
6. Click "Finish"

Repeat for each service database you need (just change the Database name in step 3).
This way each connection shows only its own database — clean and simple.
